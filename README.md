### How to compile css

`sass --style compact --watch scss:css`  
or with debug infos    
`sass --style compact --debug-info --watch scss:css`  


### How to start the local server
`cd path/to/this/project`  
`npm install`  
it will install Connect  
`node server.js`  
it will start a node server on port 3000  
http://localhost:3000
